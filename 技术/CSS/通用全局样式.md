```css
:root {
    box-sizing: border-box;
}

*,
::before,
::after {
    /** 设置所有元素的box-sizing为border-box */
    /** 用inherit，是为了方便第三方组件覆盖自己的样式 */
    box-sizing: inherit;
}

body {
    background-color: #eee;
    font-family: Helvetica, Arial, sans-serif;
}

/** 选中相同父级的非第一个元素 */
body * + * {
    margin-top: 1.5em;
}
```
